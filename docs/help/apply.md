# 申请网络资源

为满足老师和同学的科研、办公和学习需要，网络信息中心开放部分网络资源供师生申请。

- 校（院）二级域名

- 服务器托管

- 免认证 IP 地址、MAC 地址

- 云计算资源

- 工大云盘

- VPN 访问

如您有以上资源的使用需求，请参考以下流程。

在开始以下流程之前，请前往 (齐鲁工大OS平台)[https://os.qlu.edu.cn/] 。
