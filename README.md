# QLU_WLYW_Web

齐鲁工业大学（山东省科学院）网络信息中心

网络与高性能计算协会 网络运维 Web

## 部署教程

```sh
pip install -r requirement.txt
cd /path/to/this/repo
mkdocs build
mkdocs serve
```
